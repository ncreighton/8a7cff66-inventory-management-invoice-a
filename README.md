== Description ==

The Inventory Management Invoice and Payment WordPress Plugin is a comprehensive solution for e-commerce and small business owners who need to streamline their inventory tracking, invoicing, and payment processing directly from their WordPress dashboard. This plugin allows you to manage stock levels, create professional invoices, accept payments, and automatically update inventory upon sale or return. It integrates seamlessly with popular payment gateways like PayPal, Stripe, and Square, ensurin

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Inventory Management Invoice and Payment WordPress Plugin
